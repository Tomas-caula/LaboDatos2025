Para el correcto funcionamiento del codigo es necesarios la instalacion de los siguientes paquetes 
    1.numpy
    2.pandas 
    3.matplotlib 
    4.seaborn 
    5.scikit-learn
puedes hacerlo colocando la siguiente linea en tu terminal 

pip install numpy pandas matplotlib seaborn scikit-learn

siempre y cuando tengas pip instalado y actualizado 

Luego debes tener el dataset solicitado guardado con el nombre Fashion-MNIST.
